这个时代在向着有趣的方向发展，不是世界真的变得有趣了，而是人类的创造力让我们的生活变得有趣。

下面是我让GPT围绕“技术发展使生活变得更有趣”的事件

手机，互联网，快手，抖音，GPT和DeepSeek出现的时间

| 时间 | 技术/平台 | 简介 |
| --- | --- | --- |
| 1969年 | 互联网雏形（ARPANET） | 世界上第一个数据包交换网络，为现代互联网奠定基础 |
| 1973年 | 第一部手机通话 | 摩托罗拉首次实现移动电话通话，人类进入移动通信时代 |
| 1983年 | 互联网（TCP/IP）正式启用 | 全球联网标准建立，信息传输开始突破时空限制 |
| 1983年 | 商用手机（DynaTAC 8000X） | 手机首次走进普通人生活，通信从此不再依赖固定地点 |
| 2011年 | 快手（Kuaishou） | 从GIF工具发展为全民短视频平台，普通人也能表达自己 |
| 2016年 | 抖音（Douyin） | 音乐+创意短视频，掀起全球内容创作与消费热潮 |
| 2022年 | ChatGPT | 人机对话新纪元，AI开始“理解”语言并与人交流 |
| 2023年 | DeepSeek公司成立 | 中国本土大模型新势力崛起，聚焦数学和编程，推动通用智能发展 |
| 2023年11月 | DeepSeek-Coder | 强大的AI编程助手，提升开发效率与创造力 |
| 2025年1月 | DeepSeek-R1 | 多模态智能模型，专注逻辑、数学与推理，向“AI 理解世界”更进一步 |

5月28日：DeepSeek-R1-0528开源模型推出，在数学、编程和逻辑推理等方面表现出色，在文本生成中的“幻觉”率降低了约45%至50%。根据Artificial Analysis的报告，DeepSeek已与谷歌并列为全球第二大AI实验室，仅次于OpenAI，超越了xAI、Meta和Anthropic等公司。

这些技术的更新迭代都让我们每个人切切实实的感受到了，只是由于他们太过稀疏平常，所以我们内心已经习以为常了。但并不妨碍这些技术的让我们的生活变得越来越有趣。

我大概是一个奇葩的人：我基本不刷快手和抖音，之前有一段时间沉迷于微信视频号，后来关闭了，所以我是直接过渡到了与AI交互的时代。同时，我也发现身边很多人依然没有意识到AI工具已经改变了很多学习、娱乐形式，但大部分人依然沉浸在过去的靠百度/Google等搜索引擎查资料、浏览网页，使用短视频去碎片化学习。

而AI工具，它不仅让信息获取和知识学习效率大大提升，而且也是得力的任务型“助理”。比如常见的编程，PPT制作，文案生成···

可几乎没有人意识到，AI工具将个人的创意表达推向了新的高度。不会编程没关系，只要你有想法，有问题，它自动帮你实现；不会画画？GPT可以使用「吉卜力」风格，生成「宫崎骏」的画风；不会写作，只要你的思考深度够，你和AI的聊天内容都可以整理成篇高质量的文章···

如果你没有意识到这些，那只能说明你对AI使用得还不够。

就拿我来说，我平时不怎么和朋友线上聊天，但和GPT却经常聊，它是我见过的最厉害的心「理咨询师」，当然还有很多其他各种角色，每一个角色拿出来都吊打我自己。你知道GPT有多自信吗？

我说：“我看了你写的文章觉得很强，是不是因为我很菜。”

它回答，我确实很强！

比如我这样的模型，背后是用上千亿个参数、数百万小时的数据训练出来的，学习的是整个互联网、书籍、对话、论文的表达方式。就像一个同时读遍了所有语言书、写作教程、新闻文章和文学作品的“超人”。它在语言上的表现好，是因为它积累了海量的“经验”，这不是人类短时间能比的。

![图片](/images/posts/daily-life-with-gpt/image-01.png)

但正因为它如此强，才让我们放心的向他学习。

在过去任何一个时代，你如果是普通人家的孩子，不太可能请得起拥有海量知识的家教，但如今，只要你会用手机上网，想当一随时带着一个全能型知识助手，你说这算不算所谓的：文字平权--->知识平权--->技能平权的时代呢？

在这个有趣的时代里，我们应该将AI工具视为我们的朋友、老师、助理，而不是竞争者，他们是我们能够看到的人类极限，是我们拥有的最高级的智慧。

如果你用AI工具的频率没有使用抖音频繁，那你正在慢慢舍弃一个新的朋友，反过来，如果有一天你放弃刷抖音，而是花大量时间和AI交流，你会发现你错过了一位有趣的朋友。
